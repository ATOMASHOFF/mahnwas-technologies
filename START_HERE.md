## 🎨 CUSTOMIZATION QUICK TIPS

### Change Brand Color (2 minutes)
Open `css/styles.css` and find:
```css
:root {
  --c-accent: #c9a227;  ← Change this hex code
}
```

### Update Email Address (1 minute)
Find and replace:
```
contact@mahnwas.com
```

### Replace Images (5-10 minutes)
Find Unsplash URLs:
```html
<img src="https://images.unsplash.com/..." />
```
Replace with your images:
```html
<img src="path/to/your/image.jpg" />
```
